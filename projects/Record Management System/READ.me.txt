USER GUIDE

1) Download the app.
2) make one folder to make things systematic
3) then start using your app.
4) when you will start adding entries, the data file automatically created  within the folder to keep the track of the data.
5) Note : make sure the .exe file and the data file should be in same folder.

1) open the data file and start adding information in following format only.

ABC

Name
Age
Roll number
